# External Integrations

**Analysis Date:** 2026-03-20

## APIs & External Services

**Azure OpenAI:**
- Service: Azure OpenAI API for chat completion models
  - SDK/Client: `@microsoft/teams.openai` with OpenAIChatModel class (`src/app/app.ts` lines 67-72)
  - Auth: API key via `AZURE_OPENAI_API_KEY` environment variable
  - Endpoint: `AZURE_OPENAI_ENDPOINT` environment variable
  - Deployment: `AZURE_OPENAI_DEPLOYMENT_NAME` environment variable
  - API Version: "2024-10-21" (hardcoded in `src/app/app.ts` line 71)

**Microsoft Teams Bot Framework:**
- Service: Microsoft Teams messaging platform
  - SDK/Client: `@microsoft/teams.apps` (App class) and framework handlers (`src/app/app.ts`)
  - Messaging endpoint: `/api/messages` (configured in `m365agents.local.yml` line 38)
  - Registration: Via Bot Framework portal (`m365agents.local.yml` lines 34-41)

**Microsoft 365 Agents Toolkit:**
- Service: Microsoft's AI agents framework for Teams
  - Components used:
    - App framework: `@microsoft/teams.apps` - Core application container
    - AI layer: `@microsoft/teams.ai` - ChatPrompt and AI conversation handling
    - OpenAI integration: `@microsoft/teams.openai` - OpenAI model adapter
    - Common utilities: `@microsoft/teams.common` - LocalStorage for conversation history
    - API types: `@microsoft/teams.api` - MessageActivity, TokenCredentials types

## Data Storage

**Databases:**
- None configured - Application is stateless for database purposes

**Conversation Storage:**
- LocalStorage (in-memory): `@microsoft/teams.common` LocalStorage class
  - Purpose: Stores conversation history per conversation thread (`src/app/app.ts` lines 60-61, 92)
  - Key structure: `{conversation_id}/{user_id}`
  - Non-persistent: Data lost on application restart

**File Storage:**
- Local filesystem only:
  - Instructions file: `src/app/instructions.txt` loaded at startup (`src/app/app.ts` lines 15-21)
  - Deployed to: `lib/src/app/instructions.txt` (via `package.json` build script)

**Caching:**
- None external service
- In-memory conversation cache via LocalStorage

## Authentication & Identity

**Auth Provider:**
- Microsoft Entra ID (Azure AD)
  - Implementation: User-Assigned Managed Identity for service-to-service auth
  - Credential type: `ManagedIdentityCredential` from `@azure/identity` (`src/app/app.ts` lines 28-37)
  - Token Factory: Creates tokens dynamically for requested scopes (`src/app/app.ts` lines 23-38)
  - Client ID: `CLIENT_ID` environment variable
  - Configuration mode: Bot type `UserAssignedMsi` (checked in `src/app/app.ts` line 47)

**Token Management:**
- TokenCredentials interface with token factory function
  - Scope support: Dynamic scope array handling (`src/app/app.ts` line 31)
  - Tenant ID support: Optional tenantId parameter for multi-tenant scenarios
  - Token caching: Handled by Azure SDK

- `CLIENT_ID` - Microsoft Entra app client ID for bot authentication
- `CLIENT_SECRET` - Microsoft Entra app client secret
- `BOT_TYPE` - Type of bot authentication (e.g., "UserAssignedMsi")
- `TENANT_ID` - Azure tenant ID for token requests
- `AZURE_OPENAI_API_KEY` - Azure OpenAI service API key
- `AZURE_OPENAI_ENDPOINT` - Azure OpenAI service endpoint URL

**Environment profiles:**

## Webhooks & Callbacks

**Incoming:**
- POST `/api/messages` - Teams Bot Framework message webhook
  - Handled by: App framework message event handler (`src/app/app.ts` line 58)
  - Types: User messages and feedback submissions

**Message Event Handlers:**
- `message` - User chat messages (`src/app/app.ts` lines 58-100)
  - Streams response for 1-to-1 chats
  - Sends final response for group chats
- `message.submit.feedback` - Feedback reaction events (`src/app/app.ts` lines 102-105)
  - Logs feedback data to console

**Outgoing:**
- Bot Framework sends responses back to Teams via Activity responses
- AI model streaming via prompt.send() with onChunk callback (`src/app/app.ts` lines 84-86)

## Integration Flow

**Message Processing Flow:**
1. User sends message in Teams → `/api/messages` webhook receives MessageActivity
2. App loads conversation history from LocalStorage (`src/app/app.ts` lines 60-61)
3. Creates ChatPrompt with Azure OpenAI model configuration (`src/app/app.ts` lines 64-73)
4. Sends prompt to Azure OpenAI API with conversation context and instructions
5. Streams response back to Teams user (1-to-1) or sends full response (group chat)
6. Updates LocalStorage with new messages (`src/app/app.ts` line 92)
